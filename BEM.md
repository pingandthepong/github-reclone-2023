# 사용한 BEM 정리

## BEM (Block Element Modifier)

> 절대적인 것은 아닌, 일반적으로 많이 사용하는 방식

| 종류 |    이름    |      뜻       |        예시        |
| :--: | :--------: | :-----------: | :----------------: |
|  -   |   하이픈   | 일반적인 작명 |    .toggle-btn     |
| \_\_ | 언더바 2개 |  ~의 일부분   | .container\_\_item |
|  --  | 하이픈 2개 |   ~의 상태    |   .btn--success    |
